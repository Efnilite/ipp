- 7 new Gamemodes:
-- Duel / Duel 1-3 other players at the same time (unfinished)
-- Hourglass / You can only stand on blocks for 1 second (unfinished)
-- Practice / Select any types of jump you want to practice (unfinished)
-- Speed / Speed. (finished)
-- Speed Jump / Jump from platform to platform while you become faster and faster (finished)
-- Team Survival / If one person falls, everyone does (unfinished)
-- Time Trial / Get a score of 100 as fast as you can (finished)

- Introducing Sessions: (unfinished)
-- You can now use a menu to view and visit all public sessions.
-- You can freely join, spectate or leave these public sessions, and chat with participants.
-- There are also private sessions: these have a 6-digit code which players can use to join private lobbies to play with friends. If enabled, you can also invite players using a command.
-- You can create your own session, anytime you like, and pick a specific gamemode to play. You can customize the settings of this gamemode, and what players can and can't do.

- New Parkour Mode:
-- Lobby Mode / You can select a specific area in your lobby for players to parkour in which can run alongside the normal Parkour mode (unfinished)

- New Style Type:
-- Cumulative Styles / Styles which follow the same pattern of blocks each time (finished)
